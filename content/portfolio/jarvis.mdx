A 24/7 personal AI OS, not a chat wrapper. Telegram-fronted, runs as a launchd-managed daemon on macOS, powered by Claude Opus through ~150 MCP tools across Gmail (4 accounts), Google Calendar, Drive, Apple Music, Things 3, Playwright web automation, household SQLite memory, brain notes, and home control. Self-tooling — JARVIS writes its own tools at runtime when a job needs one. Bridges over localhost with Anna (my wife's assistant) and Mozart (my son's) so the three agents coordinate the household. Stack: TypeScript MCP servers, SQLite (FTS-backed brain), launchd, Anthropic API on subscription.
